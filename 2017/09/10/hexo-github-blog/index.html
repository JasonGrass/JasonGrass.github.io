hexo 常用命令记录

  • hexo clean
    清理

  • hexo g
    生成博客

  • hexo d
    部署博客到github

以上三条可以这样写：hexo clean && hexo g && hexo d

  • hexo s / hexo s -p 5566
    在本地4000(默认)/5566端口开启博客服务
