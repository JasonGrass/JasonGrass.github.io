GITEE.COM does not provide shell access. 是怎么回事。

Nov 12, 2018 | 编程 | 阅读

问题很简单，别想多了。

使用 gitee.com 时，冒出
You've successfully authenticated, but GITEE.COM does not provide shell access.

还以为 SSH 出了什么问题，其实，啥都没有。

开源中国原来有个域名：git@git.oschina.net，后来用了 git@gitee.com ，所以，把仓库的 git 地址更新一下就好了。
